### PaaS

#### Docker

1. Get Colvert Docker image:

    ```shell
    docker pull colvertproject/colvert
    ```

2. Start a Colvert container:

    ```shell
    docker run -d --name colvert --restart always --net host colvertproject/colvert
    ```
